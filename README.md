# Timeline
A jump-and-run game with an additional axis: Time.

## Compile and Run
**Timeline** utilizes the _CMake_ build system. Follow the steps below to compile and run the **Timeline** executable:
1.  Compile
    ```bash
    mkdir build
    cd build
    cmake ..
    cmake --build .
    ```
2.  Run (inside the `build` directory)
    ```bash
    ./timeline
    ```
